# I'm trying to contribute to the vxcontrol/pentagi repository, but I'm not sure how to submit a pull request.

_Submit a pull request to the vxcontrol/pentagi repository using GitHub's web interface._

## Steps

1. First, fork the vxcontrol/pentagi repository by clicking the 'Fork' button on the repository's GitHub page.
2. Next, clone your forked repository to your local machine using the command `git clone https://github.com/your-username/pentagi.git`.
3. Then, create a new branch for your changes using `git checkout -b my-feature`.
4. Make your changes and commit them using `git add . && git commit -m 'My feature'`.
5. Finally, push your changes to your forked repository using `git push origin my-feature`, and then submit a pull request through GitHub's web interface.

## Pitfalls

- Forgetting to create a new branch for your changes, which can lead to conflicts with the main branch
